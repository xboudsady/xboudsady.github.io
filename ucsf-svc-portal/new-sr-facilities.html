Event Seteveup

Please book 5 days in advance to ensure setup can be scheduled. Include the following information in the work description:

  • Event Date(s)
  • Event Time (period during which booking is actually taking place)
  • Setup/Teardown Hours (the time added to event time to allow staff to prepare a room or cleanup an event)
  • Reserved Time (total time for which the room is reserved; generally event time plus setup/teardown hours)
  • Required Features:
    • easels
    • recycling bins
    • extra trash cans
    • extra chairs
    • extra tables
  • Include setup style and for special setups include diagram.
